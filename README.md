# Java – Estudos e Fundamentos

Este repositório reúne meus estudos em Java, com foco em fundamentos da linguagem,
lógica de programação, orientação a objetos e boas práticas que estou aprendendo no curso Java10x

## Conteúdos abordados
- Variáveis e tipos de dados
- Estruturas condicionais e laços de repetição
- Métodos
- Orientação a Objetos (classes, herança, polimorfismo)
- Exercícios práticos

## Objetivo
Consolidar a base em Java e servir como apoio para projetos futuros e preparação
para oportunidades de estágio.

## Como executar
Os exemplos podem ser executados diretamente pelo IntelliJ IDEA,
utilizando o JDK configurado no projeto.
